How it works

Up and running
in four steps

01

Install deps

pip install PyQt6 pynput â€” that's the whole dependency list.

02

Run the app

python main.py opens the control panel. Pick a preset and click Show Overlay.

03

Capture in OBS

Add a Window Capture source, select Keyboard Overlay, tick Allow Transparency.

04

Customise

Drag keys, change colours, set your font. Save as a named config and switch any time.

Get it

Install

terminal 
# clone the repo
git clone https://github.com/you/keyboard-overlay
cd keyboard-overlay

# install dependencies
pip install -r requirements.txt

# run
python main.py

Requirements

Python 3.10 or newer. Works on Windows, macOS and Linux. On macOS you'll need to grant Accessibility permissions so pynput can read keystrokes.
